# 中途岛双环境部署指南

## 一、服务器端配置（需手动执行）

### 1. 创建开发环境目录和数据库

```bash
# SSH 登录中途岛
ssh root@47.113.221.26

# 创建开发环境目录
mkdir -p /home/bookbox-dev

# 创建开发数据库
mysql -u root -p
CREATE DATABASE IF NOT EXISTS bookbox_dev CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit
```

### 2. 首次部署开发环境

在本地执行：
```bash
./deploy-dev.sh
```

### 3. 配置 PM2（服务器端）

```bash
# 查看当前服务
pm2 list

# 开发环境应该会自动启动，如果没有：
cd /home/bookbox-dev
pm2 start src/index.js --name bookbox-dev

# 保存 PM2 配置
pm2 save
```

### 4. 配置 Nginx（服务器端）

编辑 `/etc/nginx/conf.d/bookbox.conf`，添加开发环境路径：

```nginx
# 生产环境（已有）
location /bookbox/api/ {
    proxy_pass http://localhost:3000/;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_cache_bypass $http_upgrade;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
}

# 开发环境（新增）
location /bookbox-dev/api/ {
    proxy_pass http://localhost:3001/;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_cache_bypass $http_upgrade;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
}
```

重启 Nginx：
```bash
nginx -t
systemctl reload nginx
```

## 二、本地开发流程

### Git 分支策略

- `main` 分支 → 生产环境（/home/bookbox + bookbox 数据库）
- `develop` 分支 → 开发环境（/home/bookbox-dev + bookbox_dev 数据库）

### iOS 编译配置

Xcode 中有两个 Scheme：

1. **BookBox-Production**（Release 配置）
   - Bundle ID: `com.bookbox.app`
   - 显示名称：书库
   - 有 App 图标
   - 连接：`http://47.113.221.26/bookbox/api`

2. **BookBox-Development**（Debug 配置）
   - Bundle ID: `com.bookbox.app.dev`
   - 显示名称：书库
   - **无 App 图标**（以作区分）
   - 连接：`http://47.113.221.26/bookbox-dev/api`

### 部署命令

```bash
# 部署生产环境（main 分支）
git checkout main
./deploy.sh

# 部署开发环境（develop 分支）
git checkout develop
./deploy-dev.sh
```

## 三、环境隔离说明

| 项目 | 生产环境 | 开发环境 |
|------|---------|---------|
| Git 分支 | main | develop |
| 服务器目录 | /home/bookbox | /home/bookbox-dev |
| 数据库 | bookbox | bookbox_dev |
| 端口 | 3000 | 3001 |
| Nginx 路径 | /bookbox/api | /bookbox-dev/api |
| PM2 进程名 | bookbox | bookbox-dev |
| iOS Bundle ID | com.bookbox.app | com.bookbox.app.dev |
| iOS 图标 | ✅ 有 | ❌ 无 |

## 四、注意事项

1. 两套环境数据库完全独立，互不影响
2. 开发环境可以随意测试，不会影响生产数据
3. iOS 测试版无图标，安装后可通过 Spotlight 搜索"书库"启动
4. 功能测试通过后，从 develop 合并到 main，再部署生产环境
