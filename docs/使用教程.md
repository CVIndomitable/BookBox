## 一、核心概念

理解以下四个概念，后面的操作就很顺了：

| 概念 | 说明 | 举例 |
|------|------|------|
| **书库 Library** | 一个独立的书籍空间。多个书库之间互相隔离。 | "家里"、"办公室"、"老家" |
| **书架 Shelf** | 书库内的活跃存放区，放平时常看的书。 | "客厅书架"、"床头柜" |
| **箱子 Box** | 书库内的归档区，放打包或暂存的书。 | "搬家 01 号箱"、"旧书储物箱" |
| **书籍 Book** | 一条书的记录。**一本书同一时间只在一个位置**：书架 / 箱子 / 未归位。 | 《三体》在"客厅书架" |

分类（小说、工具书……）是**全局共享**的，所有书库通用。

## 二、首次使用

1. 打开 App，底部三个 Tab：**扫描 · 书库 · 设置**。
2. 进入**设置**页：
   - 确认"**服务器连接**"测试通过（绿色勾）。
   - 选择**区域**：国内（豆瓣查书）或海外（Google Books）。
   - 打开"**应用内语音控制**"，屏幕会出现可拖动的麦克风悬浮按钮。
3. 进入**书库**页：
   - 顶部是书库切换条。首次使用点"**+ 新建**"创建第一个书库，例如"家里"。
   - 在总览视图中点"**新建书架**"建立第一个书架，例如"客厅书架"。
4. 回到**扫描**页，准备开始拍书。

> App 会记住上次打开的书库，下次启动自动定位。

## 三、预分类模式（快速浏览和分类）

用途：对着一堆还没整理的书拍几张照片，快速看看都有什么书、大致什么类别。**不会写入书库**，只是临时列表，适合整理前的"盘点"。

**操作步骤：**

1. 扫描页 → 点**预分类模式**（蓝色卡片）。
2. 点"**开始拍照**"或右上角 **+** 按钮。
   - **拍照**：对准一排书，拍清楚书脊或封面即可。
   - **从相册选取**：也可以直接用已有的照片。
3. 等待 1~3 秒，AI 返回识别结果列表。每本书右侧会显示**三色标签**（绿/黄/红，见第八节）。
4. 顶部有**分类筛选条**，点某个分类只看该类的书。
5. 对单本书可以：
   - **左滑删除**错误识别。
   - 点**铅笔图标**手动修改书名 / 作者 / 分类。
6. 底部按钮：
   - "**保存当前列表**"：把这次识别的结果存成一个会话，稍后能在装箱模式中一键导入。
   - "**载入已保存列表**"：调出之前保存的会话继续处理。
   - "**新建空白列表**"：清空重来。

> 预分类结果**只是临时数据**，真正要入库还得到装箱模式里操作。

## 四、装箱模式（正式录入并归位）

用途：把书**真正录入书库**，并放到某个书架或箱子里。

**操作步骤：**

1. 扫描页 → 点**装箱模式**（橙色卡片）。
2. 顶部会出现**目标位置选择条**。**必须先选好目标位置**，扫描按钮才会点亮：
   - 选一个已有的书架 / 箱子；或
   - 点"**新建**"即时建一个箱子（例如 "搬家 03 号箱"）。
3. 点右上角 **+** 按钮，三种输入方式：
   - **拍照**：现场拍。
   - **从相册选取**：导入已有照片。
   - **从预分类导入**：把之前在预分类模式保存的会话整批导入。
4. 识别完成后进入**扫描结果**页：
   - 每本书前有**勾选框**，默认全选。不想入库的可取消勾选。
   - 书名后有**三色状态标签**，可判断识别质量（第八节）。
   - 点**铅笔**改书名 / 作者 / ISBN。
5. 底部显示"已选择 N 本"，点"**入库**"。
   - 所选书会写入数据库，`location_type` 自动设为 `shelf` 或 `box`，指向刚才选的位置。
   - 目标容器的书籍计数同步 +N。
   - `book_logs` 记录一条 `add` 日志，日后可追溯。

> 想把一本书**从 A 箱移到 B 架**？不要在装箱模式做，去"书库"里进入该书详情，或直接对语音助手说"把《三体》移到客厅书架"（见第六节）。

## 五、书库管理

**书库**页是日常管理入口。

### 顶部书库切换条

- **点一下**切换当前书库。
- **长按**某个书库名弹出菜单：重命名 / 修改位置 / 删除。
- "**+ 新建**"建立新书库。

### 两种视图（上方分段控件切换）

**总览**
- 统计：总书籍数 / 未归位数 / 书架数 / 箱子数。
- 书架区：列出所有书架。点进去看其中的书，或"**新建书架**"。
- 箱子区：列出所有箱子。点进去看内容，或新建。

**全部书籍**
- 一本一条列出所有书，支持**搜索**和**分页**。
- 左滑某条可删除。
- 点一本进入详情页，查看 / 编辑 / 移动 / 看操作历史。

### 右上角 ⋯ 菜单

- **操作日志**：看全局所有书的移动 / 添加 / 删除记录。
- **分类管理**：增删分类。
- **扫描历史**：查看过往识别记录。

### 书架 vs 箱子怎么选

| 场景 | 推荐位置 |
|------|----------|
| 经常翻、随手拿的书 | 书架 |
| 打包搬家、季节性不用 | 箱子 |
| 还没想好放哪 | 暂不选位置（未归位），后续再移 |

## 六、语音助手

一个悬浮在屏幕任意位置的麦克风按钮，用自然语言操作书库。

**开启方式**

- 设置 → 打开"**应用内语音控制**"。
- 首次使用会请求语音识别权限，允许即可。

**怎么用**

1. **拖动**麦克风按钮到屏幕任意顺手的位置。
2. **点一下**麦克风图标展开控制面板。
3. 点"**开始说话**"，贴近话筒说出指令，说完点"**停止**"。
4. 面板中会显示识别出的文字 + AI 回复。如果右侧出现 ⚡ 图标，代表命中了缓存（响应更快）。

**常用指令示例**

| 你说 | App 做什么 |
|------|-----------|
| "把《三体》移到客厅书架" | 找到《三体》并移动位置，写操作日志 |
| "《活着》在哪" | 回复当前位置 |
| "客厅书架有多少本书" | 返回数量统计 |
| "新建一个书架叫卧室书架" | 创建书架 |
| "书库一共有多少书" | 返回总数 |

> 找不到书、重名、指令歧义时，AI 会追问。按提示补充一句就行。

## 七、Siri 快捷指令：查找书籍

不用打开 App，直接用 Siri 就能定位一本书。

**触发方式**（任一均可）

- "Hey Siri，在 BookBox 找书"
- "Hey Siri，BookBox 查找书籍"
- "Hey Siri，BookBox 这本书在哪"

**流程**

1. Siri 接收到触发词后会问："**你要找哪本书？**"
2. 说出书名（可模糊，比如"三体"、"活着"）。
3. Siri 朗读结果：
   - 唯一匹配：《三体》在书架「客厅书架」
   - 多本匹配：找到 3 本：《三体 I》在书架「客厅书架」；《三体 II》在箱子「搬家 02 号箱」…

> 锁屏或开车时特别好用——不用解锁 App。

## 八、三色标识怎么看

App 里到处都会看到三色圆点或标签，代表**这条书籍信息的可信度**。

**AI 识别模式（默认）**
- 🟢 **绿色 high** — AI 高置信度，一般可直接入库。
- 🟡 **黄色 medium** — AI 不太确定，建议点开看看封面、必要时手动改。
- 🔴 **红色 low** — AI 拿不准，强烈建议人工确认后再入库。

**OCR 回退模式**（AI 服务临时不可用时自动切换）
- 🟢 **matched** — 联网校验（豆瓣 / Google Books）成功匹配。
- 🟡 **uncertain** — 部分字段匹配，信息可能不完整。
- 🔴 **not_found** — 联网没查到，仅凭 OCR 原文，请人工核对。

**经验法则**：入库前把红色的那几本先改掉或删掉，黄色的扫一眼，绿色的直接放行。

## 九、常见问题

**Q：识别不准怎么办？**
- 拍照时保证书脊字清晰、光线充足、不反光。
- 一张照片尽量不要超过 10~15 本书。
- AI 识别失败时 App 会自动回退到本地 OCR，精度会降低但仍可用。
- 最终总可以手动点铅笔修改。

**Q：同一本书被录入了两遍？**
- 进入"书库 → 全部书籍"，搜索书名，左滑删除重复的那条。

**Q：一本书要从 A 书库搬到 B 书库？**
- 目前跨书库移动需要先删除再在目标书库重新录入。单书库内部（书架↔箱子↔未归位）可以用语音助手或书籍详情页直接移。

**Q：服务器连接失败？**
- 设置页点"**测试连接**"定位问题。
- 确认 Wi-Fi / 蜂窝网络正常，以及服务器（上海节点）运行中。

**Q：想看"上周都动了哪些书"？**
- 书库页右上角 ⋯ → **操作日志**。支持按动作类型（移动 / 添加 / 编辑）筛选。

---

**推荐工作流**

> 新书到手 → **预分类模式**盘点并分类 → 保存会话 → **装箱模式**选好目标位置 → "从预分类导入" → 勾选 → **入库**。平时查找用 **Siri 快捷指令**，整理移动用**语音助手**。
