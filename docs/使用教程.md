## 一、核心概念

理解以下五个概念，后面的操作就很顺了：

| 概念 | 说明 | 举例 |
|------|------|------|
| **书库 Library** | 一个独立的书籍空间。多个书库之间互相隔离。 | "家里"、"办公室"、"老家" |
| **房间 Room** | 书库内的物理分区，用来给书架/箱子分组。新建书库时自动带一个**默认房间**（不可删）。 | "客厅"、"书房"、"储藏室" |
| **书架 Shelf** | 房间内的活跃存放区，放平时常看的书。 | "客厅书架"、"床头柜" |
| **箱子 Box** | 房间内的归档区，放打包或暂存的书。 | "搬家 01 号箱"、"旧书储物箱" |
| **书籍 Book** | 一条书的记录。**一本书同一时间只在一个位置**：书架 / 箱子 / 未归位。 | 《三体》在"客厅书架" |

层级关系：**书库 → 房间 → 书架 / 箱子 → 书**。书架或箱子搬到另一个房间（甚至另一个书库）时，其中的书随之移动。

分类（小说、工具书……）是**全局共享**的，所有书库通用。

## 二、首次使用

1. 打开 App，首次启动会进入**登录 / 注册**页：
   - **注册**：填写用户名、密码（至少 8 位），邮箱和显示名可选。
   - **登录**：已有账号直接输入用户名密码。
   - 登录状态会被记住，下次启动直接进入主界面。
2. 登录后底部 Tab：**扫描 · 书库 · 设置**（如启用了查重 Tab，会多一项"**查重**"）。
3. 进入**设置**页：
   - 顶部"**当前用户**"显示已登录账号，页底有红色"**退出登录**"按钮。
   - 确认"**服务器连接**"测试通过（绿色勾）。
   - 选择**区域**：国内（豆瓣查书）或海外（Google Books）。
   - 打开"**应用内语音控制**"，屏幕会出现可拖动的麦克风悬浮按钮。
   - "**装箱**"分组里可以调整"**最近使用的箱子数量**"（1~5，默认 3）——装箱模式的选箱子页顶部会优先展示最近选过的这几个箱子。
   - "**查重**"分组两个开关："**加书时查重**"和"**显示「查重」Tab**"（见第六节）。
   - "**AI 性能**"分组可展开"**AI 缓存统计**"查看命中次数 / 未命中次数 / 命中率 / 缓存条目，下方有红色"**重置统计**"按钮。
   - 底部"**回收站**"进入被软删除的书本列表（见第七节）。
4. 进入**书库**页：
   - 顶部是书库切换条。首次使用点"**+ 新建**"创建第一个书库，例如"家里"。新书库会自动带一个"默认房间"。
   - 在总览视图里可以"**新建房间**"（例如"客厅"、"书房"），再新建书架或箱子时选择所属房间。
   - 点"**新建书架**"建立第一个书架，例如"客厅书架"。
5. 回到**扫描**页，准备开始拍书。

> App 会记住上次打开的书库，下次启动自动定位。

## 三、预分类模式（快速浏览和分类）

用途：对着一堆还没整理的书拍几张照片，快速看看都有什么书、大致什么类别。**不会写入书库**，只是临时列表，适合整理前的"盘点"。

**操作步骤：**

1. 扫描页 → 点**预分类模式**（蓝色卡片）。
2. 点"**开始拍照**"或右上角 **+** 按钮。
   - **拍照**：对准一排书，拍清楚书脊或封面即可。
   - **从相册选取**：也可以直接用已有的照片。
3. 等待 1~3 秒，AI 返回识别结果列表。每本书右侧会显示**三色标签**（绿/黄/红，见第十节）。
4. 顶部有**分类筛选条**，点某个分类只看该类的书。
5. 对单本书可以：
   - **左滑删除**错误识别。
   - 点**铅笔图标**手动修改书名 / 作者 / 分类。
6. 底部按钮：
   - "**保存当前列表**"：把这次识别的结果存成一个会话，稍后能在装箱模式中一键导入。
   - "**载入已保存列表**"：调出之前保存的会话继续处理。
   - "**新建空白列表**"：清空重来。

> 预分类结果**只是临时数据**，真正要入库还得到装箱模式里操作。

## 四、装箱模式（正式录入并归位）

用途：把书**真正录入书库**，并放到某个书架或箱子里。

**操作步骤：**

1. 扫描页 → 点**装箱模式**（橙色卡片）。
2. 顶部会出现**目标位置选择条**。**必须先选好目标位置**，扫描按钮才会点亮：
   - 选一个已有的书架 / 箱子；或
   - 点"**新建**"即时建一个箱子。新箱子必须选定**书库 + 房间**，否则"创建"按钮灰色不可点。
3. 点右上角 **+** 按钮，三种输入方式：
   - **拍照**：现场拍。
   - **从相册选取**：导入已有照片。
   - **从预分类导入**：把之前在预分类模式保存的会话整批导入。
4. 识别完成后进入**扫描结果**页：
   - 每本书前有**勾选框**，默认全选。不想入库的可取消勾选。
   - 书名后有**三色状态标签**，可判断识别质量（第十节）。
   - 点**铅笔**改书名 / 作者 / ISBN。
5. 底部显示"已选择 N 本"，点"**入库**"。
   - 所选书会写入数据库，`location_type` 自动设为 `shelf` 或 `box`，指向刚才选的位置。
   - 目标容器的书籍计数同步 +N。
   - `book_logs` 记录一条 `add` 日志，日后可追溯。
   - 若设置里开启了"**加书时查重**"，录入前会自动检测本次这批书是否已存在于当前书库（按"**书名 + 出版社**"判断），发现重复会弹出"**发现重复书籍**"对话框，最多列出前 5 本，可选择"**仍然录入**" / "**跳过重复项**" / "**取消**"。

> 想把一本书**从 A 箱移到 B 架**？不要在装箱模式做，去"书库"里进入该书详情，或直接对语音助手说"把《三体》移到客厅书架"（见第八节）。

### 给已有书架 / 箱子加书（不走装箱模式）

进入具体**书架详情**或**箱子详情**页，点右上角 **+** 按钮，两种方式任选：

- **拍照识别**：现场拍一张，AI 识别后直接录入本书架 / 本箱子。
- **挑选已有书**：从当前书库的"未归位"书籍列表里勾选几本，一次性搬进来。

这个入口适合日常增补，避免开装箱模式来回选目标位置。

## 五、书库管理

**书库**页是日常管理入口。

### 顶部书库切换条

- **点一下**切换当前书库。
- **长按**某个书库名弹出菜单：重命名 / 修改位置 / 删除。
- "**+ 新建**"建立新书库。

### 两种视图（上方分段控件切换）

**总览**
- 统计：总书籍数、未归位数。
- 房间区：列出当前书库的所有房间。点进去可管理该房间内的书架/箱子，或"**新建房间**"。默认房间不可删除。
- 书架区：列出所有书架（带房间标注）。点进去看其中的书，或"**新建书架**"。
- 箱子区：列出所有箱子（带房间标注）。点进去看内容，或从对应房间新建。

**全部书籍**
- 一本一条列出所有书，支持**搜索**和**分页**。
- 左滑某条可删除。**删除的书不会立即消失**，会先进入回收站，30 天内可还原（见第七节）。
- 点一本进入详情页，查看 / 编辑 / 移动 / 看操作历史。

### 右上角 ⋯ 菜单

- **操作日志**：看全局所有书的移动 / 添加 / 删除记录。
- **分类管理**：增删分类。
- **扫描历史**：查看过往识别记录。

### 书架 vs 箱子怎么选

| 场景 | 推荐位置 |
|------|----------|
| 经常翻、随手拿的书 | 书架 |
| 打包搬家、季节性不用 | 箱子 |
| 还没想好放哪 | 暂不选位置（未归位），后续再移 |

### 搬动书架 / 箱子

进入**书架详情**或**箱子详情**页 → 右上角 ⋯ → **编辑**，在"归属"一节切换目标**书库**和**房间**即可。其中的书会随之一起搬走，书架/箱子的所属书库自动跟新房间保持一致。

删除一个非默认房间时，其中的书架/箱子会自动转移到同书库的默认房间，不会丢。

## 六、查重

防止重复收入同一本书，有两种使用方式。判定规则是"**书名 + 出版社**"完全一致才算重复。

### 方式 A：装箱时自动检测

- 设置 → 查重 → 打开"**加书时查重**"。
- 之后每次在装箱模式或扫描结果页点"入库"前，App 都会比对当前书库的已有藏书。发现重复时弹出"**发现重复书籍**"对话框，最多列出前 5 本，可选"**仍然录入**" / "**跳过重复项**" / "**取消**"。

### 方式 B：全库查重

- 设置 → 查重 → 打开"**显示「查重」Tab**"，底部会多一个"**查重**"Tab（文档图标）。
- 点进去 App 会扫描当前书库，按"书名 + 出版社"把重复的书分组展示，同时显示重复分组数和重复书本数。
- 可在分组内选定多余副本一键删除（进回收站，可还原）。

> 两个开关相互独立：只想装箱时提醒就只开 A，只想平时偶尔清一下就只开 B。

## 七、回收站

删除不是"真·删除"，而是软删除——先进回收站，**30 天后才真正清除**。

**打开方式**：设置页底部 → **回收站**（垃圾桶图标）。

**界面说明**

- 每条记录显示书名、作者、删除时间和"N 天后清除"的倒计时。
- 页面底部 Footer 写着"**保留 30 天后自动彻底删除**"。

**单条操作**

- **左滑** → 绿色"**还原**"按钮，把书恢复回原位置（或未归位）。
- **右滑** → 红色"**彻底删除**"按钮，立即永久删除（无法再还原，会弹二次确认）。

> 误删了一整箱书？第一时间来回收站挨个还原。超过 30 天就只能重新录入了。

## 八、语音助手

一个悬浮在屏幕任意位置的麦克风按钮，用自然语言操作书库。

**开启方式**

- 设置 → 打开"**应用内语音控制**"。
- 首次使用会请求语音识别权限，允许即可。

**怎么用**

1. **拖动**麦克风按钮到屏幕任意顺手的位置。
2. **点一下**麦克风图标展开控制面板。
3. 点"**开始说话**"，贴近话筒说出指令，说完点"**停止**"。
4. 面板中会显示识别出的文字 + AI 回复。如果右侧出现 ⚡ 图标，代表命中了缓存（响应更快）。

**常用指令示例**

| 你说 | App 做什么 |
|------|-----------|
| "把《三体》移到客厅书架" | 找到《三体》并移动位置，写操作日志 |
| "《活着》在哪" | 回复当前位置 |
| "客厅书架有多少本书" | 返回数量统计 |
| "新建一个书架叫卧室书架" | 创建书架 |
| "书库一共有多少书" | 返回总数 |

> 找不到书、重名、指令歧义时，AI 会追问。按提示补充一句就行。

## 十、三色标识怎么看

App 里到处都会看到三色圆点或标签，代表**这条书籍信息的可信度**。

**AI 识别模式（默认）**
- 🟢 **绿色 high** — AI 高置信度，一般可直接入库。
- 🟡 **黄色 medium** — AI 不太确定，建议点开看看封面、必要时手动改。
- 🔴 **红色 low** — AI 拿不准，强烈建议人工确认后再入库。

**OCR 回退模式**（AI 服务临时不可用时自动切换）
- 🟢 **matched** — 联网校验（豆瓣 / Google Books）成功匹配。
- 🟡 **uncertain** — 部分字段匹配，信息可能不完整。
- 🔴 **not_found** — 联网没查到，仅凭 OCR 原文，请人工核对。

**经验法则**：入库前把红色的那几本先改掉或删掉，黄色的扫一眼，绿色的直接放行。

**书详情里手动确认校验状态**

已入库的书点进详情，"校验信息"一节可以手动调整黄色"待确认"状态：

- **点一下**黄色"待确认"徽章 → 改为绿色"已匹配"（手动认可 AI / OCR 的结果）。
- **长按**绿色"已匹配"徽章 0.5 秒 → 改回黄色"待确认"（反悔时用）。

这样无需编辑书名，也能快速把黄色的"心里没底"转成绿色的"已核实"。

## 十一、网页版与多人协作

除了 iOS App，BookBox 还有一个**网页版**，适合在电脑上批量管理、做 CSV 导入导出，以及**多人共用一个书库**。网页版用同一套账号登录（首次使用请先在 App 或网页上注册一个账号）。

> 网页版部署地址由管理员提供，若不清楚可询问部署同学。

### 功能概览

- **书库列表 / 书籍列表**：与 App 同步，搜索与筛选更方便。
- **CSV 导入 / 导出**：整书库批量搬迁、对账或备份。
- **成员管理**：把书库分享给他人协作（见下）。
- **晒书提醒**：设置定期晒书的提醒规则（见第十二节）。

### 三种角色

在网页版 → 书库详情 → **成员** Tab 里管理：

| 角色 | 权限 |
|------|------|
| **所有者 Owner** | 全部权限，可转让书库、可解散。 |
| **管理员 Admin** | 增删改书籍、书架、箱子、成员（不含所有者操作）。 |
| **成员 Member** | 查看与编辑书籍，不能改成员 / 删库。 |

**添加成员**：所有者 / 管理员点 **+ 添加成员**，填对方用户名，选择角色（成员 / 管理员）即可。右侧"**移除**"按钮把成员踢出该书库。

> iOS App 端目前**没有**成员管理界面，如需分享书库请登录网页版操作。

## 十二、晒书提醒

为了防止整箱封存的书受潮发霉，系统支持定期提醒"该把某某箱书拿出来晒一晒"。

**该功能目前只在网页版可用**（iOS 端暂无入口）。

**设置方式**：网页版顶部菜单 → "**晒书提醒**"。

**三级规则（从低到高优先生效）**：

1. **用户默认晒书间隔**（默认 365 天）——没做任何单独设置时的兜底值。
2. **按书库**覆盖——某书库可以单独设置间隔。
3. **按箱子**覆盖——某箱子可以再单独设置间隔。

**到期操作**：

- 到期的箱子会出现在提醒列表里。
- 现场把书拿出来晒完后，点"**已晒书**"按钮，倒计时按该规则重新开始。

## 十三、常见问题

**Q：识别不准怎么办？**
- 拍照时保证书脊字清晰、光线充足、不反光。
- 一张照片尽量不要超过 10~15 本书。
- AI 识别失败时 App 会自动回退到本地 OCR，精度会降低但仍可用。
- 最终总可以手动点铅笔修改。

**Q：同一本书被录入了两遍？**
- 事前预防：设置 → 查重 → 打开"**加书时查重**"，入库前会自动提示。
- 事后清理：打开底部"**查重**"Tab（需先在设置里打开"显示「查重」Tab"），一次看到全库所有重复分组，批量删除多余副本。
- 误删的可以在回收站 30 天内还原。

**Q：一本书要从 A 书库搬到 B 书库？**
- 单本书跨书库移动目前需要先删除再在目标书库重新录入。单书库内部（书架↔箱子↔未归位）可以用语音助手或书籍详情页直接移。
- 如果要**整架 / 整箱**一起搬到另一个书库，去书架详情 / 箱子详情页 → 编辑 → 归属，切书库+房间即可（书会一起跟过去）。

**Q：服务器连接失败？**
- 设置页点"**测试连接**"定位问题。
- 确认 Wi-Fi / 蜂窝网络正常，以及服务器（上海节点）运行中。

**Q：想看"上周都动了哪些书"？**
- 书库页右上角 ⋯ → **操作日志**。支持按动作类型（移动 / 添加 / 编辑）筛选。

**推荐工作流**

> 新书到手 → **预分类模式**盘点并分类 → 保存会话 → **装箱模式**选好目标位置 → "从预分类导入" → 勾选 → **入库**（开启"加书时查重"可避免重收）。平时查找用 **Siri 快捷指令**，整理移动用**语音助手**；批量管理、CSV 导入导出、分享书库给家人一起用——去**网页版**；长期封箱的书别忘了配上**晒书提醒**。
